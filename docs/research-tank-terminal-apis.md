# 储罐终端数据 API 调研报告

**调研日期：** 2026年4月
**背景：** ETI 平台需要支持储罐终端作为独立的验证对象，用于能源商品贸易合规/尽职调查。

---

## 核心结论

> **全球没有免费的综合储罐终端 API。**
> 主要终端运营商（Vopak、Oiltanking、Stolt-Nielsen）均无公开 API，数据只能通过第三方聚合平台获取。

---

## 一、免费 / 低成本选项

### EIA Open Data API
- **价格：** 完全免费
- **覆盖：** 美国境内炼油厂、管道终端、散装终端的库存数据
- **局限：** 仅限美国，无全球终端名录，无运营商执照信息
- **接入：** https://www.eia.gov/opendata/
- **ETI 适用性：** ⭐⭐ 低（地理范围太窄）

### OilPriceAPI
- **价格：** 免费层 1,000次/月；付费从 $9/月 起
- **覆盖：** 60+ 主要港口燃料价格（MGO、VLSFO、HFO），港口列表
- **局限：** 聚焦燃料定价，终端合规数据深度不足
- **接入：** https://www.oilpriceapi.com/
- **ETI 适用性：** ⭐⭐ 低（偏价格，非尽职调查）

### Global Fishing Watch API
- **价格：** 非商业免费；商业需咨询
- **覆盖：** 卫星检测的海上固定设施（油气平台）
- **局限：** 主要是近海平台，不含陆上储罐
- **ETI 适用性：** ⭐ 极低

## 二、付费选项（需联系报价）

### TankTerminals.com（Insights Global）⭐⭐⭐ 推荐
- **价格：** 7天免费试用（无需信用卡）；正式订阅需联系销售
- **数据规模：** 全球 **16,102+ 储罐终端和生产设施**
- **字段：** 容量、连接性、产品流、扩张计划、证书、后勤绩效、运营商联系信息
- **覆盖类型：** 独立终端、炼油厂、化工厂、LNG 液化厂、植物油工厂
- **接入：** https://tankterminals.com/
- **ETI 适用性：** ⭐⭐⭐ 最高（最完整的全球储罐终端数据库）
- **备注：** 也通过 Bloomberg 和 Refinitiv 分发数据

### Insights Global / ICE（PJK 数据集）
- **价格：** 企业订阅，通过 ICE Connect
- **覆盖：** ARA（阿姆斯特丹-鹿特丹-安特卫普）、富查伊拉、新加坡的每日存储水平
- **接入：** https://developer.ice.com/
- **ETI 适用性：** ⭐⭐ 中（偏市场数据，非终端名录合规）

### MapStand 油气终端数据集
- **价格：** 基础预览免费；企业版需报价
- **覆盖：** 全球油气终端、LNG 终端地理空间数据
- **接入：** https://mapstand.com/datasets-oil-and-gas
- **ETI 适用性：** ⭐⭐ 中（地理空间层有用）

## 三、企业级选项（高价，不公开定价）

### Kpler（含 MarineTraffic）⭐⭐⭐
- **核心能力：**
  - 覆盖 36,000+ 船只的实时货物追踪
  - 终端入港/离港记录，港口周转时间分析
  - 40+ 大宗商品（原油、成品油、LNG、LPG）
  - 制裁合规模块：整合 OFAC、OFSI、EU 制裁名单，AIS 欺诈检测
  - MarineTraffic Vessels API（GraphQL）：220,000+ 船只，含所有权层级、UBO
- **接入：** https://www.kpler.com/
- **ETI 适用性：** ⭐⭐⭐ 极高（船只+货物+终端合规全链路，但价格是门槛）

### Vortexa ⭐⭐⭐
- **核心能力：** 全球 22,500+ 储罐、120+ 国家、68 亿桶容量；终端级别库存追踪
- **历史数据：** 追溯至 2016 年
- **接入：** https://www.vortexa.com/
- **ETI 适用性：** ⭐⭐⭐ 高（终端级库存深度极强）

### Kayrros（已被 Energy Aspects 收购，2026年3月）
- **核心能力：** 基于 20+ 卫星星座的 AI 图像分析，追踪全球 12,000+ 原油储罐，每日更新
- **特点：** 卫星独立核查，可交叉验证申报数据
- **接入：** https://www.kayrros.com/
- **ETI 适用性：** ⭐⭐ 中高（库存异常预警，非终端名录）

### Windward AI ⭐⭐⭐
- **核心能力：**
  - Due Diligence & Compliance API（GraphQL）：船只风险评分、行为指标
  - 公司筛查（含 UBO、D&B 企业关系图谱）
  - 制裁名单监控（OFAC、OFSI、EU 等）
  - AIS 欺诈/暗中船对船转运检测
  - 港口和终端流量监控，可设置预警
- **接入：** https://windward.ai/api-hub/due-diligence-compliance-api/
- **ETI 适用性：** ⭐⭐⭐ 极高（船只合规 + 终端行为监控全栈）

### OPIS（Oil Price Information Service）
- **核心能力：** 全球最大的批发终端历史价格数据库（历史追溯至 1981 年），可验证交易定价
- **接入：** https://www.opisnet.com/
- **ETI 适用性：** ⭐⭐ 中（价格合规核查，非终端名录）

### Genscape / Wood Mackenzie
- **核心能力：** 逐罐库存数据，含纽约港、ARA 地区
- **接入：** https://developer.genscape.com/
- **ETI 适用性：** ⭐⭐ 中

## 六、ETI 平台选型建议

### 当前阶段（MVP）— 零额外成本
1. **Tavily API（已配置）** — 对储罐公司做网络情报搜索，覆盖大多数尽职调查问题
2. **预置种子数据** — 手动录入主要港口核心终端（富查伊拉、ARA、新加坡、宁波舟山等）

### 中期（产品验证后）
3. **TankTerminals.com** — 先申请 7 天免费试用评估数据质量，再决定是否订阅
4. **Windward AI** — 接入终端行为监控和合规全栈

### 长期（大客户/监管级）
5. **Kpler 或 Vortexa** — 终端级库存深度追踪，覆盖伊朗、俄罗斯等高风险路线

## 七、关键风险场景（为什么储罐终端验证重要）

典型的"洗产地"交易链：
```
伊朗 Kharg Island 装货
      ↓
阿曼海域 STS（ship-to-ship 转货）
      ↓
马来西亚某"储罐公司"接收  ← 需要验证这家公司是否真实存在、是否有实际罐容
      ↓
重新标注为马来西亚原油
      ↓
卖给中国炼厂
```

验证要点：
- 该储罐公司在声称的地点**真的有罐**吗？容量与货量匹配吗？
- 运营执照是否有效、是否在当地注册？
- 实际控制人是谁？有无制裁关联？
- 历史上是否参与过类似"洗产地"交易？
