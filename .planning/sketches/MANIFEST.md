# Sketch Manifest

## Design Direction
Professional, precise, textured dark UI in the Linear/Vercel aesthetic. Deep dark backgrounds
with layered surface hierarchy. Controls have visual weight: micro-gradient buttons, refined
focus glow on inputs, subtle inset shadows on surfaces. The goal is to feel like a tool that
costs money — not a generic dashboard.

## Reference Points
- Linear — button texture, surface hierarchy, micro-motion
- Vercel Dashboard — dark depth, precise typography, border treatments

## Themes
- `default.css` — Dark (Linear/Vercel). Primary use.
- `light.css` — Light (cool white, slightly blue-tinted). Client-facing / report contexts.
