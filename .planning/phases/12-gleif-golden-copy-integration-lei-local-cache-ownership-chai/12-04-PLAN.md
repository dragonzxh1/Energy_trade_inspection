**修改 2 — 在 scoreCompany 的 communityReputation 块末尾添加扣分逻辑：**

找到 `if (whitelisted) C = Math.min(10, C + 2)` 这一行（在 `else` 块内），在其之后、`else` 块闭合括号 `}` 之前，没有特别要求。

实际上，reportingExceptionFlag 需要在 `if (fraudHits > 0) { C = 0 }` 的 else 块之后，但在 `clamp` 之前。最安全的插入点是在整个 communityReputation 块结束后（`C` 的最终值计算完成后，`return` 语句之前）：

在 `return {` 之前（scoreCompany 函数内部），或在 C 值最终固定后，添加：
```typescript

    C = Math.max(0, C - 3)
  }
```

**具体插入位置规则：**
1. 必须在 `if (fraudHits > 0) { C = 0 } else { ... }` 块之后
2. 必须在 `return { E: clamp(E, 25), ... }` 之前
3. 使用 `Math.max(0, C - 3)` 防止负值（clamp 在此处之后执行，但提前防卫更清晰）

**完整 scoreCompany 返回前的代码应为：**
```typescript

  return {
```
