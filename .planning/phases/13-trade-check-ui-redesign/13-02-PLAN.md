**定位目标：** 找到 `function inputStyleNew` 的定义（Plan 01 中已创建占位版本）。

**精确替换为以下实现：**

```typescript
function inputStyleNew(key: string, focused: string | null, hasError?: boolean): React.CSSProperties {
  const isFocused = focused === key
  return {
    width: '100%',
    boxSizing: 'border-box' as const,
    background: 'rgba(0,0,0,0.28)',
    border: `1px solid ${
      hasError  ? 'rgba(239,68,68,0.5)' :
      isFocused ? '#6366f1' :
                  'rgba(255,255,255,0.07)'
    }`,
    boxShadow: isFocused
      ? 'inset 0 2px 3px rgba(0,0,0,0.3), 0 0 0 2px rgba(99,102,241,0.18)'
      : 'inset 0 2px 3px rgba(0,0,0,0.3), inset 0 1px 0 rgba(0,0,0,0.12)',
    borderRadius: '7px',
    color: '#f1f1f3',
    fontSize: '13px',
    fontFamily: 'inherit',
    padding: '8px 12px',
    outline: 'none',
  }
}
```

**注意签名变化：** 新函数接受 `focused: string | null` 作为第二个参数（不再是内部读 state）。

**更新 TradeForm 中的调用点：** 由于 `focused` state 存在于 `TradeForm` 内部（Plan 01 已创建），在 `field()` 辅助函数中将 `inputStyleNew(key, hasError)` 改为 `inputStyleNew(key, focused, hasError)`。
