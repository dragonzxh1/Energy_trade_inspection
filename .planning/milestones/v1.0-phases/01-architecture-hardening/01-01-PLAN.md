| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-01 | Spoofing | middleware.ts — session verification | mitigate | `auth()` validates NextAuth database session token; cannot be forged without DB access |
| T-01-02 | Elevation of Privilege | /api/admin/sync — admin role check | mitigate | `isAuthorized()` checks ADMIN_EMAILS allowlist and ADMIN_SECRET bearer token; unauthenticated = 401, authenticated non-admin = 403 |
